# Security — Glix

## Objetivo

Garantir privacidade e segurança dos dados dos usuários.

---

## Estratégia Inicial

- Supabase Auth
- HTTPS
- Row Level Security
- Controle de acesso
- Banco protegido

---

## LGPD

O Glix deverá:
- informar política de privacidade
- permitir exclusão de conta
- proteger dados pessoais
- armazenar dados com segurança

---

## Observação Importante

O Glix não realiza diagnóstico médico.
O aplicativo funciona como ferramenta de registro e acompanhamento pessoal.
