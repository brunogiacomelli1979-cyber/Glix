# Glix

## Visão Geral
Glix é um aplicativo SaaS de monitoramento e registro pessoal de glicemia, desenvolvido com foco em simplicidade, segurança, organização de dados e experiência moderna de usuário.

O objetivo do projeto é permitir que usuários acompanhem seus registros de glicemia de forma intuitiva, segura e visualmente clara.

---

## Objetivos do Projeto

- Registro simples de glicemia
- Histórico organizado
- Dashboard visual
- Relatórios em PDF
- Notificações inteligentes
- Segurança e privacidade dos dados
- Estrutura escalável para futuras funcionalidades
